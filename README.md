## Crypcoin: Real-Time Crypto Market Data Streaming

Crypcoin is a dynamic web application designed to provide users with real-time
cryptocurrency market data using Binance's WebSocket API. Leveraging the power
of Lightweight Charts, the platform delivers seamless and interactive
visualizations of live price movements, order book updates, and trade history.
Built for traders and crypto enthusiasts, Crypcoin offers a responsive and
intuitive interface to monitor market trends and make informed decisions.
